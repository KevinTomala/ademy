¿Qué es Ademy?

Ademy es un sistema web de gestión académica y administrativa para centros de capacitación. Permite gestionar el ciclo completo: desde el registro de interesados hasta la emisión de diplomas, pasando por matrículas, pagos y facturación.

Módulos principales

Módulo Descripción Estado
Seguridad Usuarios, roles, permisos, sesiones, auditoría Activo
Operativo Centros, empresas, interesados, bitácora Activo
Académico Cursos, promociones, matrículas, diplomas Activo
Financiero Pagos, bonificaciones, comprobantes Activo
RRHH Nómina, docentes, contratos, horas En desarrollo
Contífico Integración contable / facturación electrónica En desarrollo
